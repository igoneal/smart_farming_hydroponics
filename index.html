SMART FARMING

This is an Internet of Things cloud based smart Farming Research, this research will use the climatic data from the hydroponics farm for analytics and prediction

The sensors are connected to the Mobile edge computing(MEC) devices within this smart farm and these MEC are connected to the cloud.

The sensor page displays all the sensors connected to the MEC in the Hydroponics Farm

The Reading page display all the readings from each sensor connected to the MEC receiving the climatic data within the hydroponics Farm
